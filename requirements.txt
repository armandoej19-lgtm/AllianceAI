#   1. pip install torch --index-url https://download.pytorch.org/whl/cpu
#   2. pip install -r requirements.txt
#   3. pip install -e .
#
# Install order (Colab / Linux with GPU):
#   1. pip install torch                   # standard index, gets CUDA build
#   2. pip install -r requirements.txt
#   3. pip install prophet                 # optional, add after pystan builds
#
# Prophet note:
#   Prophet requires pystan, which compiles C++ on install.  On Windows this
#   often fails with the standard pip install.  Use conda instead:
#     conda install -c conda-forge prophet
#   On Linux/Colab it installs cleanly with pip.
#   If prophet is absent, AllianceAI falls back to Holt-Winters automatically.
